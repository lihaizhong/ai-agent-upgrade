# 版本历史

## v5.0.0 (2026-03-27)

### 重大更新：学习模式启发式重构

#### 学习模式优化

**启发式学习流程**
- 从"填鸭式"改为"启发式"学习理念
- 课程结束后直接进入启发式提问环节（不问"有问题吗"）
- 提供4个简单提问角度：是什么/何时用/有何区别/怎么用
- 大白话表述，降低认知门槛

**综合学习面板**
- 同时展示：启发式提问 + 可选练习 + 学习导航
- 用户自主选择下一步，降低交互成本
- 不问"有问题吗"或"做题吗"

**练习题框架**
- 动态题型 + 强约束设计
- 6种题型根据课程特点选择（识别/诊断/设计/选择/对比/组合）
- 4项强制检查确保与课程核心概念强关联

#### 考试模式优化

**考试结构**
- 选择题5道，涵盖初级/中级/高级/专家四个难度
- 填空题3-4道，中级及以上难度
- 大题2-3道，考察提示词设计和多技术融合
- 共10-12题

**评价体系**
- 即时反馈：每题答完立即告知对错
- 严厉评价：考试结束提供严厉但建设性的评价
- 深度解析：选择题/填空题150字内精要解析，大题300字内深度剖析
- 结果保存：自动保存完整考试报告到 `exam-result/` 目录

#### 文档更新

- 重写 `reference/learning-mode.md`，启发式学习架构
- 更新 `reference/exam-mode.md`，重新设计考试结构
- 更新 `SKILL.md`，同步学习模式和考试模式描述
- 更新 `evals/evals.json`，更新测试用例

## v4.0.0 (2026-03-21)

### 重大更新：交互优化 + 提示词生成模式

#### 交互优化

- **单选交互**：将手动输入「学习/考试」改为 Question 工具单选交互
- 用户体验更直观便捷

#### 新增功能：提示词生成模式

通过结构化流程帮助用户生成高质量提示词：

**核心流程**：
1. **需求澄清**：引导式提问，澄清用户真实需求
2. **方案设计**：确定技术选型和提示词结构
3. **提示词生成**：根据方案生成初稿
4. **审查优化**：自动检查简洁性、明确性、完整性、可执行性
5. **用户确认**：确认提示词满足需求

**技术特点**：
- 头脑风暴引导清单，帮助深入理解需求
- 根据需求复杂度推荐合适的技术（零样本/少样本/思维链等）
- 生成后自动审查，保证提示词简单有效
- 支持多次迭代优化直到用户满意

#### 文档更新

- 新增 `reference/prompt-generation-mode.md` 详细说明
- 更新 SKILL.md，添加三大模式入口
- 更新 description，完善技能描述

## v3.0.0 (2026-03-19)

### 重大更新：学习模式 + 考试模式双轨制

#### 新增功能

**学习模式**
- 17门系统化课程，从入门到精通
- 精简优化的课程内容，更适合教学
- 互动问答机制，通俗易懂的解释
- 智能练习提醒，巩固学习效果

**考试模式**
- 四级难度：初级、中级、高级、专家
- 随机模式：仅从中级及以上选择
- 每场考试3-5题
- 即时讲解和评分（四维度评分）
- 考试总结报告，个性化改进建议
- 历史记录对比分析

#### 课程体系

- **基础课程**：零样本提示、少样本提示
- **推理课程**：思维链、自我一致性、思维树
- **知识课程**：生成知识提示、RAG、链式提示
- **工具课程**：ReAct、PAL、ART
- **优化课程**：APE、主动提示、DSP
- **前沿课程**：自我反思、多模态CoT、图形提示

#### 文件结构优化

- 新增 `courses/` 目录存放课程内容
- 每门课程独立文件，按需加载
- 精简 SKILL.md 主文件，提升可维护性
