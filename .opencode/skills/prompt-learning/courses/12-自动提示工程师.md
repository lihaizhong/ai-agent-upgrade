# 自动提示工程师 (APE)

## 核心概念

APE 是一种**自动生成和优化提示词**的框架。让AI自己设计提示词，然后测试效果，选出最优的。

> **一句话理解**：让AI来当提示词工程师。

## 工作原理

```
任务示例
  ↓
AI生成多个提示词候选项
  ↓
在测试集上评估每个提示词
  ↓
选择效果最好的提示词
```

## 实用发现

APE 发现的优秀零样本CoT提示：

```
让我们一步步解决这个问题，以确保我们有正确的答案。
```

比原始的"让我们一步步思考"效果更好。

## 核心优势

1. **自动化**：无需人工设计
2. **数据驱动**：基于测试效果选择
3. **发现创新**：可能发现人类想不到的提示
4. **节省时间**：自动迭代优化

## 适用场景

✅ 提示词优化
✅ 零样本学习
✅ 任务迁移
✅ 批量提示设计
