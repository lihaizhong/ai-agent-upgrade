# 生成知识提示 (Generated Knowledge Prompting)

## 核心概念

生成知识提示是指让 AI 在回答问题前，**先生成相关知识**，再基于知识进行推理。就像考试前先复习相关知识，再答题。

> **一句话理解**：先想后答，激活脑海中的知识。

## 工作原理

```
问题
  ↓
生成相关知识
  ↓
整合问题 + 知识
  ↓
基于知识推理答案
```

## 实用示例

### 示例：地理判断

**问题**：
```
希腊比墨西哥大。
```

**步骤1 - 生成知识**：
```
希腊面积约131,957平方公里
墨西哥面积约1,964,375平方公里
墨西哥比希腊大约15倍
```

**步骤2 - 基于知识判断**：
```
根据知识，墨西哥面积更大
结论：该陈述错误
```

## 适用场景

✅ 需要背景知识的问题
✅ 事实验证
✅ 常识推理

## 与RAG的区别

| 生成知识提示 | RAG |
|-------------|-----|
| 激活内部知识 | 检索外部知识 |
| 不需要外部资源 | 需要知识库 |
| 适合通用知识 | 适合最新/专业信息 |
