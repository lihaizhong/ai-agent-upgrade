# 多模态思维链 (Multimodal CoT)

## 核心概念

多模态思维链将**文本和视觉信息**同时融入推理过程。不仅看文字，还要看图片，综合理解后推理。

> **一句话理解**：图文并茂，推理更强。

## 工作原理

```
文本输入 + 图像输入
  ↓
第一阶段：生成推理说明（基于图文）
  ↓
第二阶段：推断最终答案
  ↓
答案
```

## 两阶段框架

**阶段1：理性生成**
- 编码文本特征
- 编码图像特征
- 融合特征
- 生成推理说明

**阶段2：答案推断**
- 结合推理说明
- 推断最终答案

## 核心优势

1. **多模态信息**：充分利用视觉线索
2. **推理增强**：更全面的上下文
3. **可解释性**：推理过程清晰
4. **性能提升**：显著优于单模态

## 适用场景

✅ 科学问答（有图表）
✅ 图像问答
✅ 医学诊断（有影像）
✅ 数学问题（有图形）

## 性能表现

在ScienceQA基准上，超越了GPT-3.5。
