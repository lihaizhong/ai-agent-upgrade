# 方向性刺激提示 (DSP)

## 核心概念

方向性刺激提示通过一个小模型（策略LM）生成方向性刺激，并可用**监督学习或强化学习**来优化它，让它更好地引导大模型（黑盒LLM）产生目标输出。

> **一句话理解**：训练一个"提示专家"来指导大模型。

## 工作原理

```
策略LM（小模型）
  ↓
生成方向性提示
  ↓
黑盒LLM（大模型）
  ↓
生成输出
  ↓
评估输出质量
  ↓
反馈给策略LM（监督学习或RL优化）
  ↓
迭代改进
```

## 核心优势

1. **自动化优化**：可通过监督学习或 RL 自动优化
2. **方向性引导**：精确控制输出方向
3. **可定制**：可根据目标优化
4. **分离关注**：提示生成和答案生成分离

## 适用场景

✅ 文本摘要
✅ 问答系统
✅ 代码生成
✅ 对话系统
