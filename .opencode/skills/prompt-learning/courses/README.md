# 课程目录

本目录包含提示词工程的系统化课程内容，按难度分级组织。

## 课程结构

每门课程文件包含以下模块：

1. **核心概念** - 技术的本质定义
2. **工作原理** - 简化的工作流程
3. **关键要点** - 必须掌握的核心知识
4. **实用示例** - 典型应用案例
5. **适用场景** - 何时使用/不使用
6. **常见问题** - 学习者容易遇到的问题
7. **进阶路径** - 下一步学习建议

## 学习路径建议

### 新手路径
```
零样本提示 → 少样本提示 → 思维链提示 → 自我一致性
```

### 进阶路径
```
思维树 → ReAct框架 → 检索增强生成 → 链式提示
```

### 专家路径
```
自动提示工程师 → 自我反思 → 多模态思维链 → 图形提示
```

## 课程使用方式

1. 通过 SKILL.md 中的课程选择功能访问
2. 按学习路径循序渐进学习
3. 每学完一门课程进行练习巩固
