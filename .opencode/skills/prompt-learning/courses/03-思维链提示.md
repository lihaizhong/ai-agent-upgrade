# 思维链提示 (Chain-of-Thought Prompting)

## 核心概念

思维链提示是指在提示中**展示推理过程**，引导 AI 一步步思考。就像老师讲解数学题，把每一步都写出来，学生更容易理解。

> **一句话理解**：让 AI 把思考过程说出来，答案更准确。

## 工作原理

```
问题
  ↓
步骤1：分析问题
  ↓
步骤2：中间推理
  ↓
步骤3：得出结论
  ↓
最终答案
```

### 两种方式

1. **少样本思维链**：在示例中展示推理过程
2. **零样本思维链**：只需添加"让我们一步步思考"

## 关键要点

### ✅ 核心原则
1. **逐步分解**：将复杂问题分解为简单步骤
2. **展示过程**：让推理过程可见
3. **逻辑连贯**：每一步都要合理

### ⚠️ 重要发现
- 只有**足够大的模型**（如 GPT-4）才有这个能力
- 思维链是大型模型的新兴能力

## 实用示例

### 示例 1：数学问题（零样本 CoT）

**普通提示**：
```
小明有10个苹果，给了邻居2个，给了修理工2个，
然后又买了5个，吃了1个。
还剩多少苹果？
```

**输出（错误）**：11个 ❌

**思维链提示**：
```
小明有10个苹果，给了邻居2个，给了修理工2个，
然后又买了5个，吃了1个。
还剩多少苹果？
让我们一步步思考。
```

**输出（正确）**：
```
1. 小明开始有10个苹果
2. 给邻居2个：10 - 2 = 8个
3. 给修理工2个：8 - 2 = 6个
4. 又买了5个：6 + 5 = 11个
5. 吃了1个：11 - 1 = 10个
答案：还剩10个苹果。✅
```

### 示例 2：少样本思维链

**提示词**：
```
问题：奇数之和是否为偶数？数字：4, 8, 9, 15, 12, 2, 1
推理：奇数有9, 15, 1，共3个。3个奇数相加，和为奇数。
答案：False

问题：奇数之和是否为偶数？数字：15, 32, 5, 13, 82, 7, 1
推理：
```

**输出**：
```
奇数有15, 5, 13, 7, 1，共5个。5个奇数相加，和为奇数。
答案：False
```

### 示例 3：逻辑推理

**提示词**：
```
如果所有的鸟都会飞，企鹅是一种鸟，
那么企鹅会飞吗？
让我们一步步思考。
```

**输出**：
```
1. 前提1：所有的鸟都会飞
2. 前提2：企鹅是一种鸟
3. 根据前提1和2，可以推出：企鹅会飞
4. 但这与现实矛盾（企鹅不会飞）
5. 说明前提1"所有的鸟都会飞"不成立
答案：根据给定前提，企鹅会飞，但这个前提在现实中不成立
```

## 触发短语

零样本思维链常用的触发短语：

| 短语 | 效果 |
|------|------|
| 让我们一步步思考 | ⭐⭐⭐⭐⭐ |
| 请逐步分析这个问题 | ⭐⭐⭐⭐ |
| 请展示你的推理过程 | ⭐⭐⭐⭐ |
| 让我们分解这个问题 | ⭐⭐⭐ |

## 适用场景

### ✅ 推荐使用
- **数学问题**：需要计算和推理
- **逻辑推理**：需要多步推导
- **常识推理**：需要背景知识
- **复杂决策**：需要权衡多个因素

### ❌ 不推荐使用
- **简单任务**：单步就能解决
- **创作任务**：需要创意发散
- **实时任务**：对延迟敏感

## 常见问题

### Q1：为什么要展示推理过程？
**A**：
- 减少错误：每一步都可以验证
- 提高准确率：复杂问题效果显著
- 增加可解释性：知道AI怎么想的

### Q2：零样本CoT和少样本CoT哪个好？
**A**：
- **零样本CoT**：简单快速，适合大多数情况
- **少样本CoT**：更精确，适合需要特定推理方式的任务

### Q3：推理步骤越多越好吗？
**A**：不一定。关键是：
- 步骤要清晰
- 逻辑要正确
- 不要跳跃

## 自动思维链 (Auto-CoT)

当需要大量示例时，可以让 AI 自动生成：

1. **问题聚类**：将问题按类型分组
2. **选择代表**：从每组选一个代表问题
3. **自动生成**：用零样本CoT生成推理过程
4. **构建示例**：形成少样本CoT提示

## 与其他技术的对比

| 技术 | 推理展示 | 示例需求 | 适用任务 |
|------|----------|----------|----------|
| 零样本 | 无 | 0个 | 简单任务 |
| 少样本 | 无 | 2-5个 | 中等任务 |
| 零样本CoT | 有 | 0个 | 复杂任务 |
| 少样本CoT | 有 | 1-3个 | 很复杂任务 |

## 进阶路径

```
思维链提示（单路径推理）
    ↓
自我一致性（多路径投票）
    ↓
思维树（系统探索多种路径）
```
