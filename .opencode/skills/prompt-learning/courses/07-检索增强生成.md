# 检索增强生成 (RAG)

## 核心概念

检索增强生成（RAG）是指**先从外部知识库检索相关信息，再让 AI 基于检索内容生成答案**。就像写论文时查阅参考文献。

> **一句话理解**：AI + 搜索引擎 = 更准确的答案。

## 工作原理

```
问题
  ↓
在知识库中检索相关文档
  ↓
选择最相关的Top-K文档
  ↓
将文档作为上下文 + 问题
  ↓
AI生成答案（带来源引用）
```

## 核心优势

1. **知识可更新**：更新知识库即可
2. **减少幻觉**：基于真实文档回答
3. **可追溯**：提供答案来源
4. **专业性强**：接入专业知识库

## 适用场景

✅ 企业知识库问答
✅ 最新信息查询
✅ 专业领域问答
✅ 文档问答系统

## 简单实现

```python
# 伪代码示意
def rag(question):
    # 1. 检索
    docs = knowledge_base.search(question, top_k=3)
    
    # 2. 构建提示
    context = "\n".join(docs)
    prompt = f"""
    基于以下文档回答问题：
    文档：{context}
    问题：{question}
    """
    
    # 3. 生成答案
    answer = llm.generate(prompt)
    return answer
```
