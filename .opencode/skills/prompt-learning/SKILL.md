# 提示词工程学习系统

欢迎使用提示词工程学习系统！这是一个集学习、练习、考试于一体的综合性学习平台。

## 系统启动流程

**重要**：当用户触发本技能时，**必须首先询问用户选择模式**：

```
欢迎使用提示词工程学习系统！

请选择您想要的模式：

📚 学习模式 - 系统学习提示词技术，互动问答，循序渐进
📝 考试模式 - 挑战自我，即时反馈，获取能力评估

请输入「学习」或「考试」选择模式：
```

## 课程体系概览

| 类别 | 课程 | 难度 |
|------|------|------|
| **基础** | 零样本提示、少样本提示 | 入门 |
| **推理** | 思维链、自我一致性、思维树 | 中级 |
| **知识** | 生成知识提示、RAG、链式提示 | 中高级 |
| **工具** | ReAct、PAL、ART | 高级 |
| **优化** | APE、主动提示、DSP | 专家 |
| **前沿** | 自我反思、多模态CoT、图形提示 | 专家 |

课程详细内容见：[courses/](courses/README.md)

## 学习模式（简述）

用户选择「学习」后，进入学习模式：

1. 展示课程列表 → 用户选择课程
2. 展示课程内容
3. 互动问答（用户可提问）
4. 适时提醒做题

**详细说明**：[reference/learning-mode.md](reference/learning-mode.md)

---

## 考试模式（简述）

用户选择「考试」后，进入考试模式：

1. 选择难度（初级/中级/高级/专家/随机）
2. 生成 3-5 道题目
3. 逐题作答 → 即时讲解 → 即时评分
4. 考试结束 → 总评分 → 个性化建议

**难度说明**：
| 等级 | 特点 |
|------|------|
| 初级 | 单一技术，场景简单 |
| 中级 | 2-3个技术组合 |
| 高级 | 多技术组合，场景复杂 |
| 专家 | 前沿技术，需系统思维 |
| 随机 | 仅从中级及以上选择 |

**详细说明**：[reference/exam-mode.md](reference/exam-mode.md)
