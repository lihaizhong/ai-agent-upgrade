|------|------------|------|------|

| 4 | 自我反思 vs 强化学习 | C | ✅ |
| 5 | 方向性刺激提示 | B | ✅ |

**解析摘要**：
- **Q1**：少样本提示通过在提示中提供少量示例（Input-Output pairs）引导模型输出。
- **Q2**：CoT 最适合需要多步逻辑推理的复杂任务（如数学应用题），对简单任务收益不大。
- **Q3**：主动提示通过计算模型多次采样的分歧度（Disagreement）衡量不确定度，优先标注最"犹豫"的问题。
- **Q4**：Reflexion 使用语言反馈代替传统数值奖励，模型通过阅读"反思笔记"调整行为。
- **Q5**：DSP 训练小模型（策略 LM）生成提示词引导大模型，通过 RL 优化小模型。

### 第二部分：填空题（3 题，30 分）

| 题号 | 题目 | 用户答案 | 结果 |
|------|------|----------|------|
| 1 | Zero-Shot CoT 触发短语 | "让我们一步一步思考" | ✅ |
| 2 | 自我一致性得出最终答案的方式 | 投票 | ✅ |
| 3 | RAG 中检索信息的用途 | 上下文 | ✅ |

**解析摘要**：
- **Q1**："Let's think step by step"是 Zero-Shot CoT 的标志性触发词，无需示例即可激发推理能力。
- **Q2**：自我一致性通过多数投票（Majority Voting）从多条推理路径中选出最频繁的答案，过滤偶然错误。
- **Q3**：RAG 将检索到的外部知识作为上下文（Context）注入提示词，减少模型幻觉。

### 第三部分：大题（2 题，30 分）

#### 第 1 题：智能数学辅导助手系统设计（10/15 分）

**题目**：设计一个"智能数学辅导助手"系统架构，说明会使用哪些技术（至少 3 种）及其角色。

**用户回答**：
> 这题我还没想明白，仅仅想到了 CoT 思维链技术

**评价**：
- **优点**：抓住了核心解题技术——CoT 是解题演示的基础。
- **不足**：未扩展到系统层面的"反馈"和"自适应"机制。
- **建议架构**：
  | 环节 | 技术 | 作用 |
  |------|------|------|
  | 解题演示 | CoT | 展示详细推理步骤 |
  | 错误分析 | Reflexion | 分析错误原因，生成针对性反馈 |
  | 精准练习 | Active Prompt | 选出学生最不确定的题目训练 |
  | 例题引导 | Few-Shot | 提供类似例题作为示范 |

#### 第 2 题：提示词工程与 Agent 的关系（13/15 分）

**题目**：为什么说"提示词工程"是通向更高级 AI 应用（如 Agent 智能体）的桥梁？结合 ReAct 或链式提示谈谈理解。

**用户回答**：
> 因为 AI 的提示词最开始仅仅只能作为聊天/回答问题的作用！自从有了 ReAct 框架之后，AI 还增强了通知 Agent 调用工具的能力，使得 AI 真正能通过工具干活

**评价**：
- **优点**：精准抓住了核心——"从聊天到干活"，理解了 ReAct 让 AI 从纯文本交互进化到工具调用。
- **不足**：可以进一步展开 Thought-Action-Observation 循环的具体机制。
- **深度解析**：ReAct 通过 Thought（推理）→ Action（调用工具）→ Observation（工具反馈）的循环，让 AI 不仅能"想"还能"做"，这就是智能体（Agent）的雏形。

## 学习总结

### 已掌握的技能

1. **基础概念清晰**：选择填空全对，对少样本、CoT、主动提示、自我反思、DSP 等核心概念理解准确。
2. **关键术语准确**：能准确回忆 Zero-Shot CoT 触发词、自我一致性投票机制、RAG 上下文注入等细节。
3. **Agent 进化理解深刻**：准确把握了提示词工程向 Agent 进化的关键路径（ReAct 框架）。

### 需要提升的方向

1. **系统设计能力**：从"单点技术"走向"多技术组合"，尝试将 CoT + Reflexion + Active Prompt 等组合解决复杂问题。
2. **架构思维**：在设计系统时，考虑完整的流程（输入 → 处理 → 反馈 → 优化），而不仅仅是核心算法。
3. **表达深度**：大题回答可以进一步展开细节，展示更深层的理解。

### 下一步学习建议

1. **实践应用**：尝试在实际项目中使用学过的技术（如用 CoT 优化工作流、用 Few-Shot 提高输出质量）。
2. **学习 Agent 框架**：深入学习 ReAct、Tool Use、Multi-Agent 等高级主题。
3. **复习前沿课程**：如果还没学完，建议学习自我反思（Reflexion）、多模态思维链、图形提示等前沿课程。
