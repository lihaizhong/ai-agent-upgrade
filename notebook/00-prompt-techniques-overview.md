## 1️⃣ 零样本提示 (Zero-Shot Prompting)

```mermaid
mindmap
  root((零样本提示))
    核心特点
      无需提供示例
      直接下达任务指令
      依赖模型预训练能力
    技术基础
      指令调整(Instruction Tuning)
      RLHF(人类反馈强化学习)
    优势
      简单易用
      快速原型开发
      适用于常见任务
    局限性
      复杂任务表现不佳
      特定领域不准确
      输出质量不稳定
    适用场景
      简单明确的任务
      通用任务(翻译/分类/摘要)
      模型充分涵盖的任务
    技术变种
      标准零样本
      角色扮演零样本
      上下文增强零样本
    相关技术
      → 少样本提示(添加示例)
      → 思维链(添加推理步骤)
```
