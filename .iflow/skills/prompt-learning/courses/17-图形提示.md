# 图形提示 (GraphPrompt)

## 核心概念

图形提示是一种将**提示学习应用于图结构数据**的方法。在图神经网络（GNN）上应用提示学习范式，实现少样本学习。

> **一句话理解**：提示词不仅能用于文本，还能用于图数据。

## 工作原理

```
图数据（节点+边）
  ↓
图编码器：生成图嵌入
  ↓
提示生成器：生成任务特定提示
  ↓
提示融合：融合图特征和提示
  ↓
任务预测器：输出结果
```

## 核心优势

1. **少样本学习**：图数据标注成本高，提示学习更高效
2. **跨任务泛化**：同一预训练模型适应多种任务
3. **参数高效**：只更新提示参数，不更新模型
4. **灵活适应**：易于适应新任务

## 适用场景

✅ 图分类（如分子性质预测）
✅ 节点分类（如用户分类）
✅ 链路预测（如好友推荐）
✅ 知识图谱推理

## 与传统GNN对比

| 传统GNN | GraphPrompt |
|---------|-------------|
| 需要大量标注 | 少量标注即可 |
| 需要微调 | 只需提示 |
| 任务特定 | 跨任务泛化 |
