# 自动推理和工具使用 (ART)

## 核心概念

ART 是一种**零样本自动化框架**，能自动将任务分解、选择工具并生成推理步骤，无需手写示例。

> **一句话理解**：AI自动规划，自己选工具，自己执行。

## 工作原理

```
新任务
  ↓
从任务库选择相似示范
  ↓
AI总结示范模式
  ↓
自动分解任务
  ↓
生成推理步骤 + 工具调用
  ↓
执行并输出结果
```

## 核心优势

1. **零样本**：无需为每个任务写示例
2. **自动分解**：自动拆分复杂任务
3. **工具集成**：自动选择合适工具
4. **易扩展**：添加工具即可增强能力

## 适用场景

✅ 数据分析
✅ 科学研究
✅ 自动化工作流
✅ 代码生成
