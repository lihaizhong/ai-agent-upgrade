# 思维树 (Tree of Thoughts)

## 核心概念

思维树是一种**树状结构的推理方法**，让 AI 系统地探索多种可能的解题路径，并选择最优解。就像下棋时思考"如果我走这步，对手会怎么走"。

> **一句话理解**：像下棋一样，提前思考多步，选最佳走法。

## 工作原理

```
                问题
              /  |  \
           思维A 思维B 思维C
           /  \    |
        A1   A2   B1
        |         |
      评估      评估
        ↓         ↓
      保留/剪枝  保留/剪枝
```

### 三步循环
1. **生成**：产生多个候选思维
2. **评估**：判断每个思维的质量
3. **搜索**：决定保留哪些，剪掉哪些

## 关键要点

### ✅ 核心概念
1. **思维**：推理的一个步骤或想法
2. **评估**：判断思维是否有效
3. **搜索**：选择最优路径

### ⚠️ 评估等级
- **sure（肯定）**：一定能通向正确答案
- **maybe（可能）**：需要继续探索
- **impossible（不可能）**：可以剪枝放弃

## 实用示例

### 示例 1：算24游戏

**问题**：用数字 4, 6, 9, 5 算出24

**思维树过程**：

```
第一层思维：
├─ 思维A: 6 × 4 = 24 → 评估：sure ✓
├─ 思维B: 9 + 5 = 14 → 评估：maybe
└─ 思维C: 9 - 5 = 4 → 评估：maybe

发现思维A已得到24！
最终答案：6 × 4 = 24 ✅
```

如果没找到，继续探索：

```
继续思维B: 14
├─ 思维B1: 14 + 6 + 4 = 24 → 评估：sure ✓
└─ 思维B2: 14 × ...

继续思维C: 4
├─ 思维C1: 4 + 4 + 6 + ? ...
```

### 示例 2：创意写作规划

**问题**：写一篇关于AI未来发展的文章

**思维树过程**：

```
主题选择：
├─ 思维A: AI取代人类工作 → 评估：maybe
├─ 思维B: AI辅助人类创造 → 评估：sure ✓
└─ 思维C: AI的伦理问题 → 评估：maybe

选择思维B继续：

结构规划：
├─ 思维B1: 引言-现状-展望-结论 → 评估：sure ✓
└─ 思维B2: 案例分析为主 → 评估：maybe

最终方案：
主题：AI辅助人类创造
结构：引言 → 现状 → 展望 → 结论
```

### 示例 3：简化版思维树提示

**提示词**：
```
假设三位不同的专家来回答这个问题。
所有专家都写下他们思考的第一个步骤，然后分享。
然后继续写下一个步骤并分享。
只要发现有专家出错，就让这位专家离开。
最后给出正确答案。

问题：一个农夫有17只羊，除了9只以外都死了，
农夫还剩几只羊？
```

**输出**：
```
专家1：17只羊，除了9只以外都死了...
专家2：除了9只以外都死了，意思是9只活着...
专家3：让我确认一下，"除了9只外都死了"...

[专家1发现错误离开]

专家2：所以活着的羊是9只
专家3：正确，答案是9只

最终答案：9只羊
```

## 搜索策略

### 1. 广度优先搜索 (BFS)
```
逐层探索，保证找到最优解
优点：不会错过最优解
缺点：计算成本高
```

### 2. 深度优先搜索 (DFS)
```
深入一条路走到底
优点：快速找到解
缺点：可能不是最优
```

### 3. 集束搜索 (Beam)
```
每层只保留最好的K个
优点：平衡效率和质量
缺点：可能错过最优
```

## 适用场景

### ✅ 推荐使用
- **复杂决策**：需要权衡多种可能
- **创意规划**：需要探索不同方向
- **策略游戏**：需要预判和规划
- **数学难题**：需要系统性探索

### ❌ 不推荐使用
- **简单问题**：不需要复杂探索
- **实时任务**：计算时间较长
- **资源有限**：成本较高

## 常见问题

### Q1：思维树比思维链好在哪里？
**A**：
- 思维链：单路径，可能走错
- 思维树：多路径，可以回退
- 思维树成功率可达90%+，思维链约50%

### Q2：计算成本很高怎么办？
**A**：优化策略：
- 使用简化版提示（三位专家）
- 设置最大深度限制
- 尽早剪枝无效路径

### Q3：如何判断一个思维是否有效？
**A**：让AI评估：
- 是否符合逻辑
- 是否通向目标
- 是否有常识错误

## 性能对比

| 方法 | 成功率 | 成本 | 适用场景 |
|------|--------|------|----------|
| 思维链 | ~50% | 低 | 简单推理 |
| 自我一致性 | ~70% | 中 | 中等推理 |
| 思维树 | ~90%+ | 高 | 复杂推理 |
