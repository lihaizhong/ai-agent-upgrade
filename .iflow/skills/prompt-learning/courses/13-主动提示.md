# 主动提示 (Active-Prompt)

## 核心概念

主动提示是一种**智能选择示例**的方法。它计算AI对哪些问题最"不确定"，然后优先让人工标注这些问题。

> **一句话理解**：AI不确定的问题，才需要人来教。

## 工作原理

```
问题集
  ↓
AI对每个问题生成多个答案
  ↓
计算答案的不一致性（不确定度）
  ↓
选择最不确定的问题
  ↓
人工标注这些问题
  ↓
用新标注提升AI能力
```

## 不确定度计算

如果AI对同一问题给出不同答案，说明它不确定：

```
问题A → 答案：7, 7, 7, 7, 7 → 不确定度低（一致）
问题B → 答案：5, 7, 3, 8, 6 → 不确定度高（不一致）
```

优先让人标注问题B。

## 核心优势

1. **高效标注**：只标注最需要的
2. **节省成本**：减少人工工作量
3. **提升效果**：针对性的学习
4. **自适应**：根据任务特点选择

## 适用场景

✅ 少样本学习
✅ 数据标注优化
✅ 教育辅导
✅ 研究探索
