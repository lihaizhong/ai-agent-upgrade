# 自我反思 (Reflexion)

## 核心概念

自我反思是一种让AI**从错误中学习**的框架。AI完成任务后，反思哪里做错了，把反思存入记忆，下次避免同样的错误。

> **一句话理解**：失败是成功之母，AI也会反思成长。

## 工作原理

```
执行任务
  ↓
失败或表现不佳
  ↓
自我反思：哪里错了？为什么？
  ↓
存储反思到记忆
  ↓
下次任务时参考反思
  ↓
表现提升
```

## 三大组件

1. **参与者（Actor）**：执行任务
2. **评估者（Evaluator）**：评判表现
3. **自我反思（Self-Reflection）**：分析错误

## 核心优势

1. **语言反馈**：比数字奖励更细致
2. **记忆机制**：记住经验教训
3. **快速学习**：少量迭代即可改进
4. **无需微调**：冻结模型也能提升

## 适用场景

✅ 游戏智能体
✅ 代码调试
✅ 复杂决策
✅ 推理任务

## 性能表现

- AlfWorld游戏：97%完成率
- 编程任务：达到SOTA水平
