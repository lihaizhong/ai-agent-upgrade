## 提示词工程核心概念

### 什么是提示词工程

提示词工程是指通过精心设计和优化输入给 AI 的文本（提示词），以获得更准确、更相关、更有用的输出的技术和艺术。它不仅仅是"会说话"，而是理解 AI 如何理解语言，并利用这些知识来最大化 AI 的能力。

### 为什么提示词工程重要

好的提示词可以：
- 提高输出质量：让 AI 更准确地理解你的需求
- 节省时间：减少反复修改和澄清的次数
- 解锁 AI 潜能：让 AI 完成更复杂的任务
- 提高一致性：在不同场景下获得稳定的结果

### 提示词工程的核心原则

#### 原则 1：清晰明确（Clarity）

提示词应该清楚地表达你想要什么，避免模糊不清的表达。

**示例**：
- ❌ 模糊：帮我写个文档
- ✅ 清晰：帮我写一份关于 RESTful API 设计的技术文档，包含定义、特点、最佳实践和示例代码，字数约 2000 字

#### 原则 2：提供上下文（Context）

给 AI 足够的背景信息，帮助它理解你的需求和期望。

**示例**：
- ❌ 缺乏上下文：优化这段代码
- ✅ 有上下文：这是一个处理用户订单的 Python 函数，在处理大量订单时性能较差。请帮我优化它的性能，要求保持原有功能不变，并添加必要的注释。

#### 原则 3：指定输出格式（Specify Output Format）

明确告诉 AI 你希望输出是什么样的格式。

**示例**：
- ❌ 未指定格式：分析这个数据
- ✅ 指定格式：分析这个销售数据，并以 Markdown 表格形式输出，包含产品名称、销售额、同比增长率三列，按销售额降序排列

#### 原则 4：提供示例（Provide Examples）

给出好示例和坏示例，帮助 AI 理解你的期望。

**示例**：
```
请将以下产品描述转换为简洁的营销文案，每条不超过 50 字。

示例：
输入：这是一款采用最新量子点显示技术的 4K 电视，支持 HDR10+ 和 Dolby Vision，色彩还原度达到 98% DCI-P3，搭载智能语音助手，支持多种投屏方式。
输出：量子点 4K 电视，HDR10+ + Dolby Vision，98% 色域，智能语音，多重投屏。

现在请转换：
输入：[用户输入的产品描述]
```

#### 原则 5：分解复杂任务（Break Down Complex Tasks）

对于复杂任务，将其分解为多个步骤，让 AI 逐步完成。

**示例**：
```
请帮我完成以下任务，按步骤执行：

步骤 1：分析用户需求，提取关键信息
步骤 2：设计数据库表结构
步骤 3：生成 SQL 建表语句
步骤 4：编写示例查询语句

需求：[用户需求]
```

## 提示词质量评估标准

当我评估你的提示词时，我会从以下维度进行评分（1-10 分）：

1. **清晰度（Clarity）**：提示词是否清楚地表达了需求
2. **完整性（Completeness）**：是否提供了必要的上下文信息
3. **具体性（Specificity）**：是否避免了模糊和抽象的表达
4. **约束性（Constraints）**：是否明确了输出的限制条件
5. **可执行性（Actionability）**：AI 是否能够直接执行
6. **有效性（Effectiveness）**：是否可能产生高质量的输出
