## 题目生成机制

本练习题库采用**智能随机题目生成机制**，确保每次练习都能获得不同的题目，避免重复，提升学习效果。

### 生成策略

1. **随机选择**：从题库中随机抽取题目
2. **难度匹配**：根据用户水平自动调整难度（初级/中级/高级）
3. **分类覆盖**：确保题目覆盖不同的提示词技术类别
4. **避免重复**：记录已练习题目，优先推荐未练习过的题目

### 如何使用随机题目

- 直接说"给我一个练习题"或"开始练习"
- 指定难度："给我一个中级难度的练习题"
- 指定类别："给我一个关于思维链的练习题"
- 连续练习："继续给我下一个练习题"

### 题目分类

- **基础学习类**：学习提示词工程基础概念
- **提示词优化类**：分析和改进现有提示词
- **练习生成类**：生成新的练习题
- **质量评估类**：评估提示词质量
- **角色扮演类**：应用角色扮演模式
- **思维链类**：应用思维链模式
- **少样本学习类**：应用少样本学习
- **输出格式类**：指定输出格式

- **迭代优化类**：应用迭代优化模式
- **模式组合类**：组合多种提示词模式

## 初级练习

### 练习 1：基础提示词改进

**难度**：初级
**类别**：提示词优化类
**目标**：学会将模糊的提示词改进为清晰、具体的提示词

**原提示词**：
```
帮我写个函数
```

**任务**：
1. 分析原提示词存在的问题
2. 设计一个改进后的提示词
3. 说明改进的理由

**参考要点**：
- 原提示词问题：缺乏上下文、未指定功能、未说明输入输出
- 改进方向：添加功能描述、指定编程语言、说明输入输出
- 示例改进：编写一个 Python 函数，接受一个数字列表，返回列表中的最大值

### 练习 10：质量评估

**难度**：中级
**类别**：质量评估类
**目标**：学会从多个维度评估提示词质量

**任务**：
评估以下提示词的质量，从多个维度进行评分（1-10 分），并提供优化建议。

**待评估的提示词**：
```
你是个前端开发专家，帮我写一个登录页面，要用 React，支持用户名密码登录，还要记住密码功能。
```

**评估维度**：
1. 清晰度（Clarity）：提示词是否清楚地表达了需求
2. 完整性（Completeness）：是否提供了必要的上下文信息
3. 具体性（Specificity）：是否避免了模糊和抽象的表达
4. 约束性（Constraints）：是否明确了输出的限制条件
5. 可执行性（Actionability）：AI 是否能够直接执行
6. 有效性（Effectiveness）：是否可能产生高质量的输出

**参考评分**：
- 清晰度：6/10（基本清晰，但可以更具体）
- 完整性：5/10（缺乏设计规范、错误处理等）
- 具体性：6/10（有具体要求，但不够详细）
- 约束性：4/10（约束条件较少）
- 可执行性：7/10（AI 可以执行，但需要补充细节）
- 有效性：6/10（可以产生可用的代码，但可能不够完善）

当前实现：

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr
```

请在完成每个步骤后暂停，等待我的确认再继续下一步。

**难度**：高级
**类别**：上下文工程类
**目标**：学会设计策略让 AI 记住和使用历史对话信息

**任务**：
设计一个策略，让 AI 在多轮对话中记住用户的技术背景和偏好，并提供个性化的建议。

**场景**：
- 用户正在咨询技术选型问题
- 需要记住用户的技术栈、项目背景、偏好等
- 在后续对话中使用这些信息

**任务**：
1. 设计上下文管理策略
2. 设计上下文提取和存储机制
3. 设计上下文使用机制
4. 提供具体的提示词示例

**参考策略**：

**第 1 轮对话：收集信息**
```
请帮我分析技术选型。但在给出建议之前，请先了解我的背景信息：

1. 项目类型：
2. 技术栈偏好：
3. 团队规模：
4. 预算和时间限制：
5. 特殊要求：

请根据我的背景信息给出针对性的建议。
```

**第 2-N 轮对话：使用上下文**
```
背景信息摘要：
- 项目类型：[从之前的对话中提取]
- 技术栈偏好：[从之前的对话中提取]
- 团队规模：[从之前的对话中提取]
- 预算和时间限制：[从之前的对话中提取]
- 特殊要求：[从之前的对话中提取]

当前问题：[用户的新问题]

请基于以上背景信息，提供针对性的建议。如果需要更新背景信息，请先询问。
```

## 练习建议

### 如何使用这些练习

1. **按顺序完成**：建议从初级练习开始，逐步增加难度
2. **独立完成**：先自己尝试，再看参考答案
3. **多次练习**：可以重复做同一道题，每次尝试不同的方法
4. **记录过程**：记录你的思考过程和遇到的问题
5. **寻求反馈**：让 AI 评估你的答案，寻求改进建议

### 如何评估自己的答案

1. **对照参考要点**：检查是否覆盖了所有要求
2. **测试提示词**：实际运行提示词，看 AI 的输出是否满意
3. **请 AI 评估**：让 AI 评估你的提示词质量
4. **比较优化空间**：思考是否有更好的表达方式
