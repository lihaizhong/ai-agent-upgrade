# RAG 课程入口

本文件只保留课程文件入口和课程定位说明。

课程编号、名称、难度、预计时长、推荐学习起点统一维护在 [../reference/catalog.md](../reference/catalog.md)。如果课程元数据有变化，先改 `catalog.md`，再检查这里的课程说明是否仍然准确。

## 使用方式

- 需要快速看课程全貌时，优先看 [../reference/catalog.md](../reference/catalog.md)
- 已经确定要学哪一节课时，再从这里打开具体课件
- 学习路径建议以 `catalog.md` 为准，不在这里重复维护
