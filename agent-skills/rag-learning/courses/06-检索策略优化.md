## 1. 常见检索策略

### 1.1 相似度检索（Similarity Search）

```python
# 基础向量检索
results = vectorstore.similarity_search(query, k=5)

# 带分数的检索
results = vectorstore.similarity_search_with_score(query, k=5)

# 设置阈值
results = vectorstore.similarity_search_by_vector(
    embedding,
    k=5,
    filter={"category": "tech"}
)
```

### 1.2 MMR（最大边际相关性）

MMR 在保证相关性的同时增加结果多样性：

```python
retriever = vectorstore.as_retriever(
    search_type="mmr",
    search_kwargs={
        "k": 5,              # 最终返回 5 个
        "fetch_k": 20,       # 初始召回 20 个
        "lambda_mult": 0.5   # 0=最相关，1=最多样
    }
)
```

### 1.3 混合检索（Hybrid Search）

```python
from langchain.retrievers import EnsembleRetriever
from langchain.retrievers.bm25 import BM25Retriever

# 向量检索
vector_retriever = vectorstore.as_retriever(search_kwargs={"k": 5})

# BM25 关键词检索
bm25_retriever = BM25Retriever.from_documents(documents)
bm25_retriever.k = 5

# 混合检索
ensemble = EnsembleRetriever(
    retrievers=[vector_retriever, bm25_retriever],
    weights=[0.5, 0.5]  # 各占 50%
)
```

## 2. 检索参数调优

### 2.1 Top-K 调整

| Top-K | 特点 |
|-------|------|
| k=3 | 速度快，可能遗漏 |
| k=10 | 平衡，推荐默认 |
| k=20+ | 召回高，速度慢 |

### 2.2 Score 阈值

```python
# 过滤低分结果
results = vectorstore.similarity_search_with_score(query, k=10)
filtered = [r for r in results if r[1] > 0.7]
```

### 2.3 分块策略影响

| chunk_size | 适用场景 |
|------------|----------|
| 200-300 | 短问答、精确匹配 |
| 500 | 通用场景（推荐） |
| 1000+ | 长文档总结 |

## 4. 查询改写

```python
# 查询扩展
expanded_queries = [
    query,
    f"{query} 的定义",
    f"什么是 {query}",
    f"{query} 相关内容"
]

# 并行检索合并
all_results = []
for q in expanded_queries:
    results = vectorstore.similarity_search(q, k=5)
    all_results.extend(results)

# 去重合并
unique_results = deduplicate(all_results)
```
