## 1. 什么是文本向量化

文本向量化（Text Embedding）是将文本转换为固定长度向量的技术，使得语义相似的文本在向量空间中距离更近。

**为什么需要向量化？**
- 计算机只能处理数值，文本需要转为数字才能计算
- 向量可以捕捉语义信息
- 相似文本 → 相近向量 → 便于检索

## 2. 向量化发展历程

| 时代 | 技术 | 特点 |
|------|------|------|
| 词袋时代 | TF-IDF、Word2Vec | 稀疏向量、语义弱 |

| 大模型时代 | Sentence-BERT、OpenAI Embedding | 高维密集向量、最佳效果 |

## 3. 常见 Embedding 算法

### 3.1 Word2Vec

```python
from gensim.models import Word2Vec

sentences = [["我", "爱", "北京"], ["北京", "是", "首都"]]
model = Word2Vec(sentences, vector_size=100, window=5, min_count=1)

# 获取词向量
vector = model.wv["北京"]  # 100 维向量
```

**特点**：词级别、无法处理一词多义

### 3.2 BERT

```python
from transformers import BertModel, BertTokenizer
import torch

tokenizer = BertTokenizer.from_pretrained("bert-base-chinese")
model = BertModel.from_pretrained("bert-base-chinese")

inputs = tokenizer("你好世界", return_tensors="pt")
outputs = model(**inputs)

# 最后一层隐藏状态
embedding = outputs.last_hidden_state[:, 0, :]  # [CLS] 向量
```

**特点**：上下文感知、支持中文

### 3.3 Sentence-BERT

```python
from sentence_transformers import SentenceTransformer

model = SentenceTransformer("BAAI/bge-large-zh-v1.5")
embeddings = model.encode(["文本1", "文本2"])

# 计算相似度
from sklearn.metrics.pairwise import cosine_similarity
similarity = cosine_similarity([embeddings[0]], [embeddings[1]])
```

## 5. 相似度计算

```python
import numpy as np

def cosine_similarity(a, b):
    """余弦相似度"""
    return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))

def euclidean_distance(a, b):
    """欧氏距离"""
    return np.linalg.norm(np.array(a) - np.array(b))
```
