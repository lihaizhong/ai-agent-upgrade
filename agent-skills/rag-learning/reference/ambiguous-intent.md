# 模糊意图处理

这份规则优先级最高。

只要用户当前请求还不能稳定归入 `学习课程`、`实战搭建`、`专题问答` 之一，就先返回固定菜单，不要先给建议。

## 何时命中

满足任一条件就命中：

- 用户只说“想了解 RAG”“想知道 RAG 是什么”“先看看 RAG”
- 用户同时提到“学习”和“做项目/搭系统”
- 用户明确说“不知道先学还是先做”“不知道从哪开始”
- 你无法稳定判断当前最优入口

## 固定输出

命中后直接输出下面这段，不要加任何解释：

```text
你现在更需要哪一种？
1. 学习课程
2. 实战搭建
3. 专题问答
选一个，我再继续。
```

## 反例

不要这样回答：

- 先学一点，再做项目
- 我建议你先学 30 分钟再开始
- 我先给你一个折中路线

这些都属于“先给建议”，不属于菜单降级。
