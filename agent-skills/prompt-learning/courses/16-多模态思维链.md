# 多模态思维链 (Multimodal CoT)

## 核心概念

多模态思维链将**文本和视觉信息**同时融入推理过程。不仅看文字，还要看图片，综合理解后推理。

> **一句话理解**：图文并茂，推理更强。

## 工作原理

```
文本输入 + 图像输入
  ↓
第一阶段：生成推理说明（基于图文）
  ↓
第二阶段：推断最终答案
  ↓
答案
```

## 两阶段框架

**阶段1：推理依据生成**
- 编码文本特征
- 编码图像特征
- 融合特征
- 生成推理说明

**阶段2：答案推断**
- 结合推理说明
- 推断最终答案

## 核心优势

1. **多模态信息**：充分利用视觉线索
2. **推理增强**：更全面的上下文
3. **可解释性**：推理过程清晰
4. **性能提升**：显著优于单模态

## 适用场景

✅ 科学问答（有图表）
✅ 图像问答
✅ 医学诊断（有影像）
✅ 数学问题（有图形）

## 性能表现

在ScienceQA基准上，超越了GPT-3.5。

## 何时使用 / 何时不该使用

### ✅ 推荐使用
- **题目依赖图像细节**：只看文字不够
- **需要解释推理过程**：不仅要答案，还要说明依据
- **图文信息互补**：文字给背景，图像给证据

### ❌ 不推荐使用
- **任务纯文本即可完成**：引入图像只会增加成本
- **图像质量很差**：噪声太大时会误导推理
- **对速度和成本极敏感**：多模态通常更贵、更慢

## 常见误区

- 把图像只当附件，没有把视觉线索真正纳入推理。
- 先下结论，再从图里“找证据”，导致解释失真。
- 图片和文字冲突时，没有显式说明以谁为准。

## 练习建议

找一张带图表的新闻截图，尝试设计一个多模态 CoT 提示，让模型先提取图表关键信息，再回答“趋势是什么、证据在哪里”。
