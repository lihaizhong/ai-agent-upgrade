# 主动提示 (Active-Prompt)

## 核心概念

主动提示是一种**智能选择示例**的方法。它计算AI对哪些问题最"不确定"，然后优先让人工标注这些问题。

> **一句话理解**：AI不确定的问题，才需要人来教。

## 工作原理

```
问题集
  ↓
AI对每个问题生成多个答案
  ↓
计算答案的不一致性（不确定度）
  ↓
选择最不确定的问题
  ↓
人工标注这些问题
  ↓
用新标注提升AI能力
```

## 不确定度计算

如果AI对同一问题给出不同答案，说明它不确定：

```
问题A → 答案：7, 7, 7, 7, 7 → 不确定度低（一致）
问题B → 答案：5, 7, 3, 8, 6 → 不确定度高（不一致）
```

优先让人标注问题B。

## 核心优势

1. **高效标注**：只标注最需要的
2. **节省成本**：减少人工工作量
3. **提升效果**：针对性的学习
4. **自适应**：根据任务特点选择

## 适用场景

✅ 少样本学习
✅ 数据标注优化
✅ 教育辅导
✅ 研究探索

## 何时使用 / 何时不该使用

### ✅ 推荐使用
- **标注预算有限**：只能挑一小部分样本做人类标注
- **问题分布复杂**：不同问题难度差异明显
- **想快速提升效果**：优先处理模型最拿不准的样本

### ❌ 不推荐使用
- **样本量极小**：不确定度估计本身就不稳定
- **人工标注很便宜**：全部标注反而更直接
- **任务答案高度主观**：不一致未必等于模型不会

## 常见误区

- 把“回答分歧”直接等同于“最有价值的样本”。
- 只看不确定度，不看样本是否代表真实业务分布。
- 标完一轮后不重新估计不确定度，导致后续选择失真。

## 练习建议

试着判断下面哪类样本更值得优先标注：
1. 模型答案高度一致，但整体都错
2. 模型答案分歧很大，且场景常见
3. 模型答案分歧很大，但场景极少出现
