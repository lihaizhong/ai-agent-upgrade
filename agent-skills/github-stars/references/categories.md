# Stars 分类规则

## 类别定义

### AI/ML
- **关键词**: ai, ml, llm, gpt, machine-learning, deep-learning, neural-network, nlp, computer-vision, transformer, rag, agent, chatbot, langchain, ollama, openai, anthropic
- **说明**: 人工智能、机器学习、大语言模型、AI Agent、RAG 等相关项目

### 开发工具
- **关键词**: cli, tool, devtools, editor, ide, plugin, extension, linter, formatter, debugger, terminal, productivity, developer-experience
- **说明**: 开发者工具、CLI 工具、IDE 插件、代码质量工具等

### 前端开发
- **关键词**: react, vue, angular, svelte, nextjs, nuxt, frontend, ui, component, css, tailwind, webpack, vite, javascript, typescript, html
- **说明**: 前端框架、UI 组件库、构建工具、CSS 框架等

### 后端开发
- **关键词**: api, server, backend, framework, database, orm, graphql, rest, microservice, go, python, java, rust, node, express, django, fastapi, spring
- **说明**: 后端框架、API 工具、数据库工具、微服务架构等

### DevOps
- **关键词**: docker, kubernetes, k8s, ci-cd, deployment, monitoring, observability, infrastructure, terraform, ansible, devops, cloud, aws, gcp, azure
- **说明**: CI/CD、容器化、基础设施即代码、监控、云平台工具等

### 数据科学
- **关键词**: data-science, data-analysis, data-visualization, pandas, numpy, jupyter, etl, data-pipeline, analytics, dashboard, chart
- **说明**: 数据分析、可视化、数据处理、ETL 工具等

### 安全工具
- **关键词**: security, pentest, hacking, encryption, vulnerability, scanner, audit, auth, oauth, jwt, cryptography
- **说明**: 安全扫描、渗透测试、加密、认证授权等

### 学习资源
- **关键词**: tutorial, course, book, learn, awesome, guide, roadmap, cheatsheet, educational,科普
- **说明**: 教程、课程、学习路线图、科普项目、awesome 列表等

### 效率工具
- **关键词**: note, task, todo, automation, workflow, productivity, timer, calendar, email, markdown, obsidian, notion
- **说明**: 笔记、任务管理、工作流自动化、效率提升工具等

### 其他
- **说明**: 无法归入上述类别的仓库

## 分类优先级

1. 首先匹配 `topics` 中的关键词
2. 其次匹配 `description` 中的关键词
3. 最后参考 `language` 辅助判断

如果匹配到多个类别，选择最具体的那个。
