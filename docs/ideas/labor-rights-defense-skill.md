# 中国劳动法维权（互联网白领）Skill（方向 1 + D）

## Problem Statement
如何帮助互联网白领在遭遇违法辞退/欠薪/加班费等纠纷时，基于“仅政府/立法/司法机关官方发布的法条与政策原文”，快速得到可执行的下一步行动清单，并生成可提交的仲裁材料草稿（非律师意见）。

## Recommended Direction
做“分诊 + 证据时间线 + 官方依据引用 + 仲裁申请书草稿”的闭环：

- 通过结构化问答把案情落到可判定的要素
- 输出证据缺口与行动顺序（下一步行动清单）
- 将已确认信息填充到仲裁文书模板，产出可提交的草稿

所有关键结论必须绑定到官方原文条款，并可回溯：

- 官方来源（URL/机构）
- 发布/生效/修订日期（如官方提供）
- 抓取时间
- 原文片段定位（例如条/款/项）

## Key Assumptions to Validate
- 官方来源可稳定抓取与更新，并能定位到条款级引用（含发布/生效/修订信息）
- 用户愿意提供足够事实与证据（否则只能输出泛化清单，价值不足）
- “模板化 + 官方引用”能显著提升用户行动效率，而不需要大量律师自由裁量

## MVP Scope
- 只覆盖 2 个场景：违法辞退、加班费/欠薪
- 地区策略（B，省必填、市可选）：分诊开始前先确认省份；市级信息作为可选增强项；省/市只影响“流程与入口信息”，不影响法条依据；若无法定位到市级官方页面则降级到省级通用指引
- 输入：10-20 个分诊问题 + 证据清单上传/勾选（不要求原件，仅描述与摘要）
- 输出：行动清单（按先后顺序）+ 证据缺口清单/时间线 + 仲裁申请书草稿（请求事项/事实与理由/证据目录/附件清单）+ 每段对应的官方条款引用

## Official Sources (Initial)
只信任政府/立法/司法机关官方发布原文，且要求可回溯。

- 法条依据（law_sources）：国家法律法规数据库 `https://flk.npc.gov.cn/index`
- 《中华人民共和国劳动合同法》页面（seed）：`https://flk.npc.gov.cn/detail?id=2c909fdd678bf17901678bf74d7106b3&fileId=&type=&title=%E4%B8%AD%E5%8D%8E%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD%E5%8A%B3%E5%8A%A8%E5%90%88%E5%90%8C%E6%B3%95`
- 流程与入口信息（procedure_sources）：只允许从种子页点击扩散到 `*.gov.cn`，禁止搜索扩散

## Not Doing (and Why)
- 不承诺胜诉、不做胜率预测：降低误导与合规风险
- 不引用任何第三方法条库/第三方解读：满足“只信官方”的数据要求
- 不做代提交、代理、律师式结论：控制责任边界
- 不扩展到“所有劳动纠纷”：避免 MVP 变成泛法律问答器

## Open Questions
