# 提示词分析报告

## 原始提示词
"帮我写个函数来处理数据"

## 问题诊断

这个提示词存在以下主要问题：

### 1. **过于模糊**
- "处理数据"是一个极其宽泛的概念，可以指：
  - 数据清洗（去除空值、重复值、异常值）
  - 数据转换（格式转换、类型转换）
  - 数据聚合（分组、统计、求和）
  - 数据过滤（筛选特定条件的数据）
  - 数据排序
  - 数据合并/连接
  - 数据验证
  - 等等...

AI 无法猜测你具体想要什么，所以无法提供有针对性的答案。

### 2. **缺少关键信息**
一个好的提示词应该包含：
- ✗ **编程语言**：Python? JavaScript? Java? Go?
- ✗ **输入数据格式**：JSON? CSV? 数组? 对象?
- ✗ **输出数据格式**：你期望得到什么样的结果？
- ✗ **具体的处理逻辑**：你想要对数据做什么？
- ✗ **使用场景**：这个函数将在什么环境下使用？
- ✗ **性能要求**：是否需要处理大量数据？
- ✗ **错误处理**：需要处理哪些异常情况？

### 3. **缺少上下文**
- 这是什么项目的一部分？
- 数据从哪里来？
- 处理后的数据要去哪里？
- 是否有现有的代码需要集成？

### 4. **缺少约束条件**
- 是否有特定的库或框架要求？
- 是否需要遵循特定的编码规范？
- 是否有性能或内存限制？

## 改进建议

### 建议的提示词结构（CRO框架）

```
Context（上下文） + Request（请求） + Options（选项）
```

### 改进示例 1：基础改进
```
请用Python编写一个函数来处理CSV数据，包括：
- 输入：CSV文件路径
- 功能：去除空行、转换日期格式、计算平均值
- 输出：处理后的数据和统计信息
```

### 改进示例 2：详细改进
```
我需要用Python编写一个数据处理函数。

上下文：
- 项目是一个数据分析工具，需要处理用户上传的CSV文件
- 数据量大约在1万-10万行之间
- 需要与现有的Flask API集成

需求：
1. 读取CSV文件
2. 清洗数据：去除重复行、处理缺失值
3. 转换数据类型：将字符串日期转换为datetime对象
4. 计算统计信息：平均值、中位数、标准差
5. 返回清洗后的数据和统计报告

要求：
- 使用pandas库
- 包含错误处理（文件不存在、格式错误等）
- 添加类型注解
- 包含文档字符串和示例用法
```

### 改进示例 3：极简但明确
```
用Python写一个函数，读取CSV文件，去除空行，返回pandas DataFrame
```

## 提示词设计最佳实践

### 1. **明确具体**
❌ 模糊："处理数据"
✅ 明确："读取CSV文件，去除空行，计算平均值"

### 2. **提供上下文**
❌ 无上下文："写个函数"
✅ 有上下文："在Flask API中，需要处理用户上传的CSV文件"

### 3. **指定输出格式**
❌ 无输出要求："处理数据"
✅ 明确输出："返回清洗后的pandas DataFrame和统计报告"

### 4. **包含约束条件**
❌ 无约束："写个函数"
✅ 有约束："使用pandas库，包含类型注解，添加错误处理"

### 5. **提供示例数据（如果复杂）**
```python
# 输入数据示例：
data = [
    {"name": "Alice", "age": "25", "salary": "50000"},
    {"name": "Bob", "age": "", "salary": "60000"},
    {"name": "Alice", "age": "25", "salary": "50000"}
]
```

## 通用提示词模板

对于编程任务，可以使用以下模板：

```
我需要用[语言]编写一个函数来完成[具体任务]

上下文：
- 项目类型：[Web应用/数据分析/机器学习等]
- 数据来源：[API/文件/数据库等]
- 使用场景：[描述具体场景]

需求：
1. [需求1]
2. [需求2]
3. [需求3]

技术要求：
- 使用库/框架：[列举]
- 性能要求：[如果有]
- 错误处理：[需要处理的异常]

输出：
- 函数代码
- 使用示例
- 测试用例（可选）
```

## 总结

原始提示词"帮我写个函数来处理数据"的问题在于：
1. 缺少具体的目标
2. 缺少必要的上下文信息
3. 没有明确的输出要求
4. 没有约束条件

改进的关键是：**让AI明确知道你想要什么，而不是让AI去猜测**。即使是简单的请求，也应该包含语言、功能、输入输出这三个核心要素。

记住：提示词的质量直接决定了输出质量。你提供的信息越具体、越清晰，AI给出的答案就越有用。
