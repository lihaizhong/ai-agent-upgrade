📖 什么是零样本提示？

零样本提示是指不给AI任何示例，直接通过自然语言指令让它完成任务。

💡 核心原理

当模型足够强大时（如GPT-4、Claude等大语言模型），
即使没有示例，也能理解和执行各种任务。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📝 基础模板

"请完成[任务描述]"

示例：
"请将以下英文翻译成中文：Hello, world!"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 适用场景

1. 简单直接的任务
2. 模型能力足够强时
3. 需要快速响应的场景
4. 任务难以提供有效示例时

❌ 不适用场景

1. 复杂或模糊的任务
2. 需要特定格式输出
3. 领域知识要求高
4. 需要保持风格一致

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💪 实践技巧

1. 清晰明确
   ✗ 模糊："帮我写点东西"
   ✓ 明确："写一封100字的产品介绍邮件"

2. 提供上下文
   ✗ 不足："分析这个数据"
   ✓ 充分："作为数据分析师，分析月度销售数据"

3. 指定格式
   ✗ 模糊："给出建议"
   ✓ 明确："给出3条建议，每条不超过20字"

4. 设置约束
   ✗ 无约束："写一首诗"
   ✓ 有约束："写一首五言绝句，主题是春天"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ 常见问题

Q: 零样本和少样本有什么区别？
A: 零样本不给示例，少样本会提供1-3个示例。

Q: 什么时候用零样本？
A: 任务简单明确、模型能力足够强时。

Q: 零样本效果不好怎么办？
A: 尝试少样本或思维链提示。
