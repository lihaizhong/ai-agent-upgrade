# 第11题参考答案：零样本 vs 少样本生产决策框架

## 关键因素（4个）

1. **任务熟悉度** — 模型是否处理过类似任务
2. **延迟预算** — p99 延迟要求（毫秒级还是秒级）
3. **成本预算** — token 消耗限制
4. **结果稳定性** — 是否允许结果波动

## 决策框架

| 条件 | 选择 |
|------|------|
| 任务常见 + 延迟敏感 | 零样本 |
| 任务常见 + 延迟宽松 | 零样本 or 少样本 |
| 任务陌生 + 结果重要 | 少样本（加示例） |
| 成本敏感 | 零样本 |

## p99 延迟说明

p99 延迟 = 99% 请求响应时间都在此值以内。
比平均值更有意义，因为容易被异常值拉偏。
